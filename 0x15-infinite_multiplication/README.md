## Prompt

Write a program in `C` that multiplies two base-10 positive numbers and prints the result followed by a new line.

If the number of arguments is incorrect or are not composed only of digits, print Error, followed by a new line, and exit with a status of 98. The only C standard library functions allowed are `malloc`, `free` and `exit`.
